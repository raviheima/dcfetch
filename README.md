<p align="center">Neofetch like rich presence for Discord</p>

## About

- only build-time dependency is a C compiler
- works on both Linux and macOS
- can be configured to show the currently playing media

## Configuration

The config file is located at `~/.config/discord-fetch`

Fore more information about the config file click [here](./CONFIG.md)

## Installing

```
./build.sh
sudo ./build.sh install
```

After that just run `dcfetch` while Discord is open
